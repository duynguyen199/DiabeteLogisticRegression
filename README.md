# Diabetes Logistic Regression Project

This project implements logistic regression analysis to predict the likelihood of diabetes onset based on various health metrics.

## Overview
The aim of this project is to build a logistic regression model using a dataset that includes various features related to health metrics, which could indicate the likelihood of diabetes.

## Features
- Patient health metrics
- Age
- BMI
- Blood pressure
- Glucose levels

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/duynguyen199/DiabeteLogisticRegression.git
   ```
2. Navigate to the project directory:
   ```bash
   cd DiabeteLogisticRegression
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the main analysis file:
   ```bash
   python diabetes_analysis.py
   ```
